你喜欢什么，擅长什么，要什么，最后要放弃什么

你的职业规划到底是什么，你现在所处的领域是不是你想要的，你喜欢什么？你擅长什么？你要什么？最后要放弃什么？这是每个人都值得思考的几个问题。

好的事情当成一种锻炼，不好的事情当成一种磨练

别人问你去年晋升失败，做了哪些变化

  • 最大的变化是对自己的思考：认识自己；把不好的事情当做一种磨练，提升自己；

物来顺应，未来不迎，当时不杂，既过不恋

遇事不要抱怨，顺其自然，对于还未发生的事不去担忧，专注做好眼前事，过去了的事就不要多想

修心

“情商低”的人如何混职场？https://mp.weixin.qq.com/s/jHR7rDmwMB0oJ73R5pZRhg

确实，当我觉得自己情商低的时候，会自责，无奈，怨恨自己。这些情商低的表现为：不会来事，不会开玩笑，跟老板说话没有话题，只会呆板的死扎在工作里….

文中告诉我，世俗眼里的“情商高”可能是别人天生的，你学不来。你要做的就是

  • 做自己，保持自己踏实价值观
  • 结果导向，用成绩说话
  • 工作中主动，保持与自己有紧密利害关系的人的良好沟通。不需要迎合，不需要与他们成为铁哥们，只需要真诚，解决问题、提供帮助的真诚之心
