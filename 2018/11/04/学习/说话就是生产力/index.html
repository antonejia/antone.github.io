1. 讲述

讲述是讲事实

问题：

  • 时间短
  • 内容零散

    没有重点；口头禅多

  • 说的都是个人看法：

    我怎么样怎么样. 没有信服力

讲述的要求

  • 说话要完整：时间不多不少
  • 要有次序
  • 只讲事实
  • 提供细节

2. 叙述

叙述是讲一个完整的故事。与讲述的区别是，讲述是指讲一个片段，片段是完整的就可以了，而且讲述不讲观点

叙述 = 讲述 + 观点

要求：

  • 构建完整的故事
  • 概括出自己的观点

筛选事实：

叙述是为了最终表达自己的观点，所有事实应该围绕着自己的观点来展开。所以，一些不利的事实要过滤掉

3. 表达

配合肢体语言的发言，栩栩如生的说话

4. 讨论

如何提问：可以针对事实，也可以针对结论。要提供细节，问细节。

要在对方的发言中听到事实。如果听不到事实就提问，然后听对方的结论。如果听不到结论也要提问，如果你不同意他的结论，还可以问对方发言中的逻辑关系。这就是讨论要过的关

5. 汇报

说话体现的是思维的敏捷性

汇报要给出实际可行的建议

6. 陈述

陈与述的结合就是比较郑重地将重要事实、观点讲究次序的表达出来。陈述经常用于总结性发言和阶段性发言。
用词偏书面词汇，逻辑系统结构更加完整和有条理，有较强的说服力

陈述 = 书面文字 + 表达。
提高说话能力，就是在提高影响力。
陈述的训练主要靠读书（形成思想）和写日志（书面训练）

要求

  • 次序。系统化
  • 层次。
